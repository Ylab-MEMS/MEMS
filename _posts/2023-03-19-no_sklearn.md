明明用了pip安装sklearn，为什么报错呢？

<!-- more -->
你是不是使用了
``` bash
pip install sklearn
```
来安装sklearn的呢？如果是的话，那么你就会遇到这个问题。

查看版本，发现sklearn的版本0.0.post1。这是为什么呢？

查看官网，发现sklearn的安装包名是`scikit-learn`，而不是`sklearn`。所以，正确的安装方法是：
``` bash
pip install scikit-learn
```
安装完成后，再次`import sklearn`，问题解决。
